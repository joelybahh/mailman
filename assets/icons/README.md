Place your release icons here before running bundling.

Required filenames:
- `32x32.png`
- `128x128.png`
- `128x128@2x.png`
- `icon.ico` (Windows)
- `icon.icns` (macOS)

If you prefer generating from a single source icon, use Tauri CLI:

```bash
npx @tauri-apps/cli icon ./branding/app-icon.png
```

Then copy the generated files into this folder.
